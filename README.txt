Dear user of the web-app Music@Napier,

Enclose please find some helpful explanatory notes:
(Start with 1.)

1. Create a database with the example data by writing them into the data directory.
   So please enter: Python create_db.py

2. Run main.py with Python

3. Login with the following user credentials:
	username = admin
	password = default

4. There are five possibilities, how to continue:
	- Filter by artist by entering a name of an artist
	- Filter by genre by entering a genre
	- Search a title, artist or genre
	- Click Overview to come back to all songs
	- Logout

5. Logout
